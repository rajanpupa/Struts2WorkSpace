# Struts 2 Starter

* Create a Dynamic web project
* Add the jar files
	* Struts2 core
	* XWork 
	* OgNL
	* commons fileupload
	* commons io
	* commons logging
	* commons logging api
	* freemarker
	* javassist
	* commons lang
* Create a user library and add all of these jar files.
* configure eclipse to place these libraries to web-inf/lib folder when build
